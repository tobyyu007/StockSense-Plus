# StockSense+

### Setup
Dependencies:
- langchain
- streamlit
- streamlit-chat
- pypdf
- chromadb
- fastembed

```bash
pip install langchain streamlit streamlit_chat chromadb pypdf fastembed
```

### Launch StockSense+
```bash
streamlit run app.py
```
